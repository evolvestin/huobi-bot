# Huobi Analytics Bot

## Описание проекта
Программный комплекс, предназначенный для автоматизированного сбора, обработки и дистрибуции аналитических данных криптовалютного рынка посредством интерфейса мессенджера Telegram. Архитектура решения реализует модульную структуру, включающую подсистему веб-скрейпинга на базе headless-браузера для генерации визуальных аналитических отчетов (графиков волатильности), механизм интеграции с облачными сервисами Google (Drive, Sheets) для синхронизации и хранения медиа-активов, а также систему управления базами данных SQLite для администрирования реестра пользователей. Функционал обеспечивает предоставление оперативной информации о рыночных трендах и индексе настроений (Fear & Greed) в рамках заданных временных интервалов.
